# Python Basics

## Variables

Python uses dynamic typing. Variables are created when you assign a value.

```python
name = "Alice"
age = 30
```

## Data Types

- **Strings**: Text enclosed in quotes
- **Integers**: Whole numbers
- **Floats**: Decimal numbers
- **Lists**: Ordered, mutable collections
- **Dictionaries**: Key-value pairs

## Control Flow

### If Statements

```python
if x > 0:
    print("positive")
elif x < 0:
    print("negative")
else:
    print("zero")
```

### Loops

```python
for i in range(5):
    print(i)

while condition:
    do_something()
```

## Functions

```python
def greet(name):
    return f"Hello, {name}!"
```
